# 图片目录说明

本目录用于存放参赛文章中的截图和示例图片。

## 需要添加的图片列表

### 1. 安装相关图片

- `01-download-project.png` - GitHub 仓库下载界面截图
- `02-copy-to-skills.png` - 将 captcha-copilot 文件夹复制到 TRAE skills 目录的截图
- `03-install-dependencies.png` - 终端中安装依赖的截图

### 2. 使用相关图片

- `04-auto-invoke.png` - AI 自动调用 captcha-copilot 技能的截图
- `05-manual-invoke.png` - TRAE 中调用 captcha-copilot 的对话框截图
- `06-env-config.png` - .env 文件配置截图

### 3. 代码示例图片

- `07-image-captcha.png` - 处理图形验证码的代码和结果截图
- `08-slider-captcha.png` - 处理滑动验证码的动图或截图

## 添加图片指南

1. 按照上述文件命名规范保存图片
2. 更新参赛文章中的图片引用
3. 提交图片到 Git 仓库

## 图片格式建议

- 截图使用 PNG 格式
- 动图使用 GIF 格式
- 图片宽度建议：800-1200px
- 确保图片清晰可读

## 临时占位符

在正式图片添加前，参赛文章中使用以下格式作为占位符：
```markdown
![图片描述](images/图片文件名.png)
```
